Technical Guidance

We offer expert assistance for handling technical issues in all education portals:

  • Login and password recovery
  • Data upload and correction support
  • Portal navigation and usage training
  • Error resolution and backend support
  • Guidance on new portal updates and functionalities

Documents Commonly Required for Technical Support:

  • Portal User ID and Registered Mobile Number
  • School Code or UDISE Code
  • Relevant Error Screenshot or Printout
  • Principal’s Aadhar Card
  • Photograph (if ID proof needed)
  • Samagra ID (if student data issue)
  • Caste Certificate (if related to reserved category issue)
  • Income Certificate (if applicable)
  • Authorization Letter (for third-party handling)

Please ensure all information provided is accurate for faster resolution.
