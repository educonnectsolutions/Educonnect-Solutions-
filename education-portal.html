  • Student Aadhar Cards
  • Class-wise Enrollment List
  • Staff Details (Excel format)
  • School UDISE Code
  • Infrastructure Photos
  • Affiliation Certificate (if applicable)
  • Recent Passport-sized Photograph
  • Samagra ID (for all students)

Ensure all documents are accurate and up-to-date for smooth processing.
